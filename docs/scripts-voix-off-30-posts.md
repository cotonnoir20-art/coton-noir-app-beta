# Coton Noir — 30 scripts voix-off (mot pour mot)

**Voix de marque** : une seule narratrice, tutoiement, chaleureux, expert accessible, jamais culpabilisant.  
**Rythme** : phrases courtes, pauses sur les hooks, énergie modérée (pas « publicitaire criard »).  
**Lancement cible** : 19 mai 2026.

**App (mai 2026)** : messages « ton profil / ta longueur / tes recettes » alignés avec repères longueur, problématiques et favoris recettes en prod — voir `docs/navigation-v2.md`.

## Import Google Sheets

1. Ouvrir [Google Sheets](https://sheets.google.com) → Fichier → Importer.  
2. Choisir `docs/calendrier-editorial-30-posts.csv` (séparateur : virgule).  
3. Ajouter une colonne **Script voix-off** : copier-coller depuis ce fichier post par post, ou importer un second onglet.

**Colonnes CSV** : `N°` · `Date publication` · `Funnel` · `Hook` · `Écran app` · `CTA` · `Statut tournage` · `Notes tournage`

**Statuts suggérés** : À tourner · En tournage · Montage · Planifié · Publié
